# Tasking of Mars Rover

## Happy Path
[] 绘制火星车的地图，火星对象，Mars
[] 接收第一行的命令，初始化火星地图区域
[] 构建一个火星车对象(绘制火星车到地图上)，MarsRover
[] 接收第二行的命令, 初始化火星车的位置
[] 让火星车执行命令：L/R/M
